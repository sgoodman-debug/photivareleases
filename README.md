# Photiva Releases

Download the latest version of **Photiva** for macOS.

👉 [Download latest release](https://github.com/sgoodman-debug/photivareleases/releases/latest)

## Requirements
- macOS 14.0 or later

## Installation
1. Download `Photiva-x.x.x.dmg`
2. Open the DMG and drag Photiva to your Applications folder
3. Launch Photiva from Applications
